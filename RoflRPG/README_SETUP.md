# RoflRPG 1.0.0 (REBUIlT)

## Build
mvn clean package

Jar: target/roflrpg-1.0.0.jar

## Install
Put jar into plugins/, start server once, edit YAML in plugins/RoflRPG/, reload with /rpgadmin reload


## Skills hotbar (Z)
- Server can't see the real Z key. We use **Swap Hands** action.
- In Minecraft controls, bind **Swap Hands** to **Z**.
- Put skills with trigger `Z` into your class skill list. Hotbar slot 1..9 = skill #1..#9.
